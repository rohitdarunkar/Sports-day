## Project Overview

This project demonstrates how callback functions can be used to execute tasks sequentially.  
It simulates a sports day with multiple events and calculates scores dynamically.

The events include:
- Opening Ceremony  
- 100M Race  
- Long Jump  
- High Jump  
- Award Ceremony  

All events run in sequence using JavaScript callbacks.

---

## Features

- Uses `setInterval` and `setTimeout` for asynchronous behavior  
- Generates random race results  
- Takes user input using `prompt()`  
- Updates and displays scores after every event  
- Declares final winners based on scores
